Abstract

Temporal Action Localization (TAL) aims to identify the start and end times of actions in untrimmed videos. In this work, we introduce PLOT-TAL, a novel multi-prompt framework that leverages Optimal Transport (OT) to enhance generalization and temporal precision in few-shot TAL tasks. By integrating OT with prompt learning, PLOT-TAL effectively aligns video features with action prompts, improving localization accuracy even with limited training data. Our approach demonstrates significant improvements on benchmark datasets THUMOS14 and EPIC-Kitchens-100, showcasing its robustness and effectiveness in real-world scenarios.

Existing methods learn a single prompt to locate and classify actions. PLOT-TAL uses multiple complementary prompts to improve class generalization and temporal discrimination. Green frames indicate foreground features.

Overview: (A) Sample overlapping video segments. (B) For each class, initialize N learnable prompt vectors concatenated with the class label. (C) Extract video features with a pre-trained 3D CNN (I3D) and prompt features via the CLIP text encoder. (D) Downsample temporally. (E) Compute optimal transport plans between prompt features and segments per temporal level; sum N vectors per class. (F) Compute cosine similarity, threshold to retrieve candidates, and regress temporal boundaries. Only the fire-marked components are trained; others are frozen.

PLOT-TAL outperforms existing methods on THUMOS14 and EPIC-Kitchens-100, achieving state-of-the-art results in few-shot TAL.
